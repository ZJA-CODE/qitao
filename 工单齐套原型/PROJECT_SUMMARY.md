## 核心算法实现

### 1. 工单优先级排序
```javascript
优先级排序：HIGH > MEDIUM > LOW
优先级相同时：按起始MRP日期升序
无优先级时：直接按起始MRP日期排序
```

### 2. 可用库存缺料数计算
```javascript
第一优先级：配置库存数量 - 未发料数量
后续优先级：上一行缺料数 - 本行未发料数量
负数累计显示，红色字体标识
```

### 3. 齐套数计算
```javascript
若工单缺料数量 >= 0：
  齐套数 = 未发料数量 / 单位用量 * 产出率

若工单缺料数量 < 0：
  齐套数 = (工单缺料数量 + 未发料数量) / 单位用量 * 产出率
```
