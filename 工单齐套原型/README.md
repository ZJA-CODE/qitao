# 工单齐套管理系统原型

基于《集团MC工单齐套表单字段模版取值逻辑.xlsx》需求文档开发的原型系统。

## 系统功能

### 配置管理
- **C1 齐套规则计算配置表** - 配置齐套计算参数（组织、分类码、仓位、供应商、工单状态）
- **C2 工单齐套优先级维护表** - 维护工单优先级
- **C3 组件责任方配置** - 配置组件责任方

### 业务查询
- **M1 工单齐套明细** - 核心业务表，显示所有工单的齐套明细
- **M2 工单详情** - 查看工单详细信息
- **M3 物料详情** - 按物料查看齐套情况

### 供给信息
- **T1 SRM PO交期报表** - 采购订单信息
- **T2 PR信息表** - 采购申请信息
- **T3 替代料明细** - 替代料信息

## 核心算法

### 工单优先级排序
- HIGH > MEDIUM > LOW
- 相同优先级按MRP日期排序

### 可用库存缺料数
- 第一优先级：配置库存数量 - 未发料数量
- 后续优先级：上一行缺料数 - 本行未发料数量

### 齐套数计算
- 工单缺料数量 >= 0：齐套数 = 未发料数量 / 单位用量 × 产出率
- 工单缺料数量 < 0：齐套数 = (工单缺料数量 + 未发料数量) / 单位用量 × 产出率

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖

## 使用方法

1. 直接用浏览器打开 `index.html`
2. 配置管理 → C1齐套规则计算配置表
3. 业务查询 → M1工单齐套明细
